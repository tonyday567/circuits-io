# circuits-io

Concrete IO transports for the circuits ecosystem:

- **`Circuit.Socket`** — TCP / WebSocket duplex
- **`Circuit.Step`** — iteration combinators on `Trace Either`
- **`Circuit.Time`** — sleep, timestamps, gaps

Free REPL dual ends (`commit` / `emit`), turns, multi-agent channels and
sessions live in **[`circuits-repl`](https://github.com/tonyday567/circuits-repl)**.
